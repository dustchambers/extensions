# Show Me File Data Changelog

## [Initial Version] - 2025-02-13

- Comprehensive file metadata viewer for macOS Finder-selected files
- File stats: name, size, type, permissions, dates, inode
- macOS Spotlight metadata categorized into Content, Dates, Media, Location, Document, Source
- EXIF data for images: camera, lens, exposure, aperture, ISO, focal length, flash, white balance
- GPS coordinates with links to Apple Maps, Google Maps, OpenStreetMap, Bing Maps
- Image color average displayed as colored tag
- Image preview resized to 300px for fast loading
- Quick Look thumbnails in file list view
- Audio/Video metadata: duration, bit rate, channels, sample rate
- Configurable sidebar sections via Raycast preferences
- Copy all metadata as JSON
- View raw metadata as formatted JSON
